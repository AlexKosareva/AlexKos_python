# Проект автоматизации тестирования с интеграцией Allure (Урок 10)

Данный проект содержит автоматизированные тесты для веб-приложений (Калькулятор и Интернет-магазин), реализованные с использованием паттерна **Page Object Model**. В проект интегрирована система отчетности **Allure** для детального анализа результатов выполнения тестов.

## Основные требования и стек технологий
*   **Язык программирования:** Python 3.14+
*   **Фреймворк для тестов:** Pytest
*   **Инструмент автоматизации:** Selenium WebDriver
*   **Отчетность:** Allure Report
*   **Менеджмент драйверов:** Webdriver-manager

## Настройка окружения
Перед запуском убедитесь, что у вас установлены необходимые зависимости:
```bash
pip install selenium pytest allure-pytest webdriver-manager
